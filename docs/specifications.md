## Description

This project is going to be two pieces that work together to form the projekt.  

We are planning to implement a mesh loader that will be able to take files of the OBJ (object) type, convert them to our own voxel format, and load them into the renderer. 

The renderer is the second part that will take the loaded file as voxels, display it and be able to raytrace the object. 

## List of features

- Minimum viable product
  - Be able to interpret a OBJ file or other 3D file type
  - Be able to load the interpreted file into the renderer
  - Be able to display voxels in different colors
- Should have
  - Be able to raytrace specular voxels
  - Move around in environment
  - Saving and loading files
- Nice to have
  - Raytracer +, more materials( glass, roughness, water )
  - GPU renderer instead of CPU
  - Basic world generation
- A finished project
  - Raytracer ++, Real time, complex raytracer (god rays, etc)
  - Real time changes to the environment
  - complex world generation
- This would break reality
  - physics, (sand, water simulation)
  - explosions
  - black holes
- Reality broken!
  - animals (with AI evolution)
